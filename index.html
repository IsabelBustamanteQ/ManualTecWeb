<a>

Permite crear un hipervínculo a cualquier URL.

  • Contenido Transparente

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido de Fraseo
  • Contenido Interactivo
  • Contenido Palpable
 

<a href="https://www.mozilla.com">Mozilla</a>

Resultado:

Mozilla

<abbr>

Representa una abreviación.

Puede contener el atributo title para colocar el significado.

Contenido Permitido:

  • Contenido de Fraseo

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido de Fraseo
  • Contenido Palpable
 
 
<p> <abbr title="Laugh Out Loud">LOL</abbr> big time.</p>

Resultado:

LOL big time.

<address>

Indica una información de contacto.

Contenido Permitido:

  • Contenido Fluido (sin address, encabezados o contenido se sección)

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido Palpable
 

<address>
    <a href="mailto:jim@rock.com">jim@rock.com</a>
</address>

Resultado:

jim@rock.com

<area>

Define un area dentro de una imagen que tiene un area oredefinida para hacer click.

Se usa dentro de un elemento <map>

Contenido Permitido:

  • Ninguno

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido de Fraseo
 

<map name="primary">
    <area
        shape="circle"
        coords="75,75,75"
        href="left.html"
        alt="Click to go Left" />
    <area
        shape="circle"
        coords="275,75,75"
        href="right.html"
        alt="Click to go Right" />
</map>
<img
usemap="#primary"
src="https://via.placeholder.com/350x150"
alt="350 x 150 pic" />

Resultado:

<article>

Representa una composición autónoma en un documento que está destinado a ser reutilizado de forma independiente.

Contenido Permitido:

  • Contenido Fluido

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido Seleccionado
  • Contenido Palpable
 

<article class="day-forecast">
    <h2>03 March 2018</h2>
    <p>Rain.</p>
</article>

Resultado:

03 March 2018

Rain.

<aside>

Representa una parte de un documento cuyo contenido está sólo indirectamente relacionado con el contenido principal del documento.

Se presentan con frecuencia como barras laterales o cuadros de notas.

Contenido Permitido:

  • Contenido Fluido

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido Seleccionado
  • Contenido Palpable
 

<article>
    <p>
        Contenido principal.
    </p>
    <aside>
        <p>Contenido relacionado al principal.</p>
    </aside>
</article>

Resultado:

Contenido principal.

Contenido relacionado al principal.

<audio controls src="/media/SaveYourTears.mp3">

<b>

Sirve para llamar la atención del lector sobre el contenido (Palabras Clave).

Antes se usaba para poner texto en negrita, pero no debe usarse para aplicar estilo al texto.

Contenido Permitido:

  • Contenido de Fraseo

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido de Fraseo
  • Contenido Palpable
 

<p>
    This article describes several <b class="keywords">text-level</b> elements. It
    explains their usage in an <b class="keywords">HTML</b> document.
</p>

Resultado:

This article describes several text-level elements. It explains their usage in an HTML document.

<base>

Especifica la URL base que se utilizará para todas las URL relativas de un documento.

Solo puede existir un elemento base en un documento.

Contenido Permitido:

  • Ninguno

Categorías a las que pertenece:

  • Contenido de Metadatos
 

<base target="_top" href="https://example.com/" />

<bdi>

Le dice al algoritmo bidireccional del navegador que trate el texto que contiene de forma aislada del texto circundante.

Es útil cuando se inserta texto dinámicamente.

Contenido Permitido:

  • Contenido de Fraseo

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido de Fraseo
  • Contenido Palpable
 

<bdi class="name">Evil Steven</bdi>: 1st place

Resultado:

Evil Steven: 1st place
Informacion adicional
<bdo>

Anula la direccionalidad actual del texto, de modo que el texto que contiene se representa en una dirección diferente.

Contenido Permitido:

  • Contenido de Fraseo

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido de Fraseo
  • Contenido Palpable
 

<p>This text will go left to right.</p>
<p><bdo dir="rtl">This text will go right to left.</bdo></p>

Resultado:

This text will go left to right.

This text will go right to left.

<blockquote>

Indica que el texto adjunto es una cita ampliada.

Contenido Permitido:

  • Contenido Fluido

Categorías a las que pertenece:

  • Contenido Fluido
  • Contenido Palpable
  • Sectioning Root
 

<blockquote cite="https://datatracker.ietf.org/doc/html/rfc1149">
    <p>
        Avian carriers can provide high delay, low throughput, and low altitude
        service. The connection topology is limited to a single point-to-point path
        for each carrier, used with standard carriers, but many carriers can be used
        without significant interference with each other, outside early spring. This
        is because of the 3D ether space available to the carriers, in contrast to
        the 1D ether used by IEEE802.3. The carriers have an intrinsic collision
        avoidance system, which increases availability.
    </p>
</blockquote>

Resultado:

Avian carriers can provide high delay, low throughput, and low altitude service. The connection topology is limited to a single point-to-point path for each carrier, used with standard carriers, but many carriers can be used without significant interference with each other, outside early spring. This is because of the 3D ether space available to the carriers, in contrast to the 1D ether used by IEEE802.3. The carriers have an intrinsic collision avoidance system, which increases availability.
